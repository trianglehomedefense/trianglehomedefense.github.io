TRIANGLE HOME DEFENSE - GITHUB PAGES UPLOAD

1. Extract this ZIP.
2. In the GitHub repo, click "uploading an existing file".
3. Drag the extracted CONTENTS into GitHub, not the ZIP folder itself.
4. Confirm index.html, CNAME, robots.txt, sitemap.xml, and assets/ are visible at the repo root.
5. Click Commit changes.
6. Go to Settings > Pages and set Source to Deploy from branch, main, /root.
7. Custom domain should be trianglehomedefense.com.

Important:
- CNAME already contains trianglehomedefense.com.
- This package is built for GitHub Pages, not Square/Squarespace editing.
- Keep the repository public unless you have GitHub Pages enabled for a private repo plan.
