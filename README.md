# Triangle Home Defense Website v2

Static GitHub Pages-ready site files for Triangle Home Defense.

## Files

- `index.html` — home page
- `wildlife-removal.html` — wildlife inspection page
- `bat-exclusion.html` — bat inspection/exclusion page
- `inspection-exclusion.html` — exclusion/access correction page
- `contact.html` — inspection request/contact page
- `assets/css/styles.css` — full visual system and responsive layout
- `assets/js/main.js` — mobile navigation and current-year footer
- `assets/brand/triangle-home-defense-mark.svg` — approved-style clover mark placeholder
- `assets/brand/favicon.svg` — favicon
- `assets/brand/social-preview.svg` — Open Graph preview image
- `robots.txt`, `sitemap.xml`, `CNAME` — deployment support

## Deployment

Copy these files into the GitHub repository root and commit. For GitHub Pages, publish from the root of the main branch unless your repository uses a different deployment setting.

## Logo note

The current website shield/tree mark should not be used. These files use a simple SVG mark in the approved clover direction. If a final production SVG or PNG logo exists, replace:

- `assets/brand/triangle-home-defense-mark.svg`
- `assets/brand/favicon.svg`
- `assets/brand/social-preview.svg`

Then verify the header, footer, favicon, and Open Graph image.

## Design direction

The site uses a dark graphite/garnet palette, off-white field-paper content sections, subtle texture overlays, readable body copy, real HTML text, and the current site structure.
